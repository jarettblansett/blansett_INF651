About This App

This Music Playlist App was created as my final for Front-End Web Development 1! It demonstrates key JavaScript concepts such as event handling, DOM manipulation, and dynamic interactivity to enhance the users experience.

Features

  • Add songs with title, artist, and optional URL
  • Mark songs as favorites
  • Search and filter songs dynamically
  • Delete/Clear songs from the playlist
